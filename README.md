#  Tic Tac Toe Game
A simple and interactive Tic Tac Toe game built with HTML, CSS, and JavaScript. This project allows two players to take turns marking spaces in a 3×3 grid, aiming to get three of their marks in a row—horizontally, vertically, or diagonally.

##  Features
*  Two-player gameplay (X and O)

*  Reset button to restart the game anytime

*  Automatic win/draw detection

*  Clean UI

##  How to Play
* Click on an empty cell to place your mark (X or O).

* The game automatically checks for a win or draw after each move.

* Click the Reset button to start a new game.
